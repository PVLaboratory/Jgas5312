    Python과 dictionary

  • Python의 기본 데이터 형태들
    • Python의 정수, 실수, 논리값 그리고 문자열은 그 자체가 값으로써 절대 바꿀 수가 없다.
    • Python은 여러 종류의 값을 묶은 값의 종류로 네 가지가 있다.
      1. 리스트: [와 ]으로 순서에 따라 여러 개를 묶었지만 필요에 따라 바꿀 수 있는 데이터
      2. 투플(tuple): (와 )으로 순서에 따라 여러 개를 묶었지만 절대 구성요소를 바꿀 수 없는 데이터
      3. 딕(dictionary): {와 }으로 순서없이 여러 개의 key:value 쌍을 묶은 데이터. 필요에 따라 바꿀 수 있는 데이터
      4. 세트(set): {와 }으로 순서없이 중복하지 않는 여러 개를 묶었지만, 절대 구성요소를 바꿀 수 없는 데이터
  • dictionary의 예제
    • 한 예로, 매년 수 많은 학생들의 국어/수학/과학/사회 등의 여러 과목을 담당하던 교사가 어느날 이름이 홍길동인 학생의 사회 과목의 성적을 알고 싶다고 하자.
    • 만약 아래와 같이 python의 리스트를 이용하여 성적을 아래와 같이 적었다고 하자.
      • 홍길동 = [60, 95, 50, 85]
    • 교사가 리스트에 적힌 성적에 상응하는 과목의 순서를 기억하고 있다면, 별다른 문제가 없을 것이나, 그렇지 않다면 리스트에 적힌 숫자들은 단순히 숫자들뿐 전혀 도움이 되지 못할 것이다.
    • 만약 국어는 95점, 수학은 85점, 과학은 60점, 사회는 50점이라고 적어 둘 수 있다면 위 문제는 전혀 문제가 되지 않을 것이다.
    • 바로 dictionary가 해답이다.
      • 홍길동 = {"국어": 95, "수학": 85, "과학": 60, "사회": 50}
    • 한 발 더 나아가, 사회 과목의 중간고사와 기말고사의 성적이 각각 70과 30이라고 하자. 그래서 위의 내용을 다음과 같이 적었다고 하자:
      • 홍길동 = {"국어": 95, "수학": 85, "과학": 60, "사회": {"중간": 70, "기말": 30}}
      • 이제 저장된 성적들이 혼란없이 명확하면서도 보다 많은 정보를 얻을 수 있을 것이다.
  • dictionary
    • 본 저자가 알고 있는 수학과 교수님이 하신 말씀 하나를 먼저 언급하고 싶다.
      • "Python을 배우기 전에 여러가지 프로그램 언어를 사용하였지만 (교수님의 전공은 응용 수학으로 코딩에 관해서는 두말할 필요가 없는 분) Python의 dictionary를 알게된 이후 Python에 푹 빠졌다."
    • 이 형태의 데이터의 구성요소는 key와 value라는 2 object가 쌍으로 key:value 처럼 콜론(:) 기호로 묶여 있다.
      • 각 구성요소는 쉼표로 구분된다.
      • key 값은 절대 바꿀 수 없는 값이어야 한다.
        • 리스트는 key값이 될 수 없으나, tuple은 key 값이 될 수 있다.
      • 빈 key값의 구성요소를 추가할 수 없으나, key값이 분명하면 None의 value 값을 가진 구성요소를 추가할 수 있다.
      • key 값은 바꿀 수 없지만 value는 언제든지 바뀔 수 있다.
      • key 값을 바꾸려면 새로운 key의 구성요소를 추가한 후, key:value의 구성요소를 삭제하는 것으로 대신할 수 있다.
      • 이 형태의 데이터는 구성요소의 순서를 따지지 않는다.

  • dictionary 형태의 값 구성요소들의 정렬
    • >>>
    

    
>>>
    

    

    
>>>
    

    

    
>>>
    d = {'bananas': 2, 'apple': 4, 'pear': 1, 'orange': 3}
    
# key 값 기준 내림차순 정렬
    
sorted(d.items(), key = lambda t: t[0], reverse = True)
    
[('pear', 1), ('orange', 3), ('bananas', 2), ('apple', 4)]
    
# value 값 기준 오름차순 정렬
    
sorted(d.items(), key = lambda t: t[1])
    
[('pear', 1), ('bananas', 2), ('orange', 3), ('apple', 4)]
    
# 문자열 key 값의 글자수 기준 오름차순 정렬
    
sorted(d.items(), key = lambda t: len(t[0]))
    
[('pear', 1), ('apple', 4), ('orange', 3), ('bananas', 2)]
    • 위 코드에서 한 가지 유념해서 살펴볼 것은 계산 결과는 더 이상 dictionary가 아니다.
      • Python의 built-in function sorted()을 사용한 결과는 항상 리스트가 된다.
    • 위 코드에 사용된 lambda는 익명 함수 (anonymous function)을 뜻하는 Python의 키워드 중 한 가지이며 추후 함수에 관한 설명에서 상세히 살펴보자.
      • 여기서는 코드에 적힌 내용을 참고하여 사용법을 연습해 보는 것을 권한다.
