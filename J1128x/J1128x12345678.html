Python 설치

  • Python 설치
    • 여기서 제공하는 정보는 The Python Software Foundation (PSF)에서 제공하는 Python 소프트웨어 설치에 관한 것
    Python 관련 제 3자가 제공하는 소트트웨어 (예: Anaconda 등등)에 관해서는 각자 구한다

    여러분의 PC 실제 화면은 여기 제공된 사진 내용과 다를 수 있음.
    여기 주어진 방법은 그동안 수 백명 학생들에게 제공한 방법으로
    제대로 따라하지 않은 일부 학생들을 제외한 대부분은 별 문제없이 Python을 사용할 수 있었다.
    그 일부 학생들도 나중에는 아래 순서를 따라하여 모두 성공하였음을 미리 밝힌다.

    1. www.python.org 방문

    2. 버튼 "Downloads"에 마우스를 올려 넣거나 클릭

    3. 버튼 "Python 3.11.0"을 클릭
        4. 본 화면 제작 당시(2022.11.22 )의 모습이므로 이후에는 Python version이 다를 수 있으나, 무조건 최신 것 사용
    4. 해당 Python 설치 파일이 모두 내려받을 때까지 기다린 후, 해당 파일을 더블 클릭

      • PC/OS에 따라 Microsoft에서 확인된 앱이 아니라는 화면이 뜨는 경우 버튼 "설치 진행"을 눌러 계속 진행한다.
    5. Python 설치 첫 화면으로 아래 사진처럼 "Add Python.exe to PATH" 를 클릭해서 checkbox 선택

    6. 중요: "Customize installation" 선택

    7. 다음 화면에서는 버튼 "Next" 클릭

    8. 중요: 마지막 화면에서 "Install for all users"의 checkbox 선택
      • 해당 checkbox를 선택 유/무에 따라 Python 설치 장소가 달라지는 것을 확인할 수 있다. (개인적으로 바꾸는 것은 절대 권하지 않음)

    9. 버튼 "Install"을 클릭

    • Setup Progress의 화면을 거쳐 최종적으로 아래 사진처럼 성공했다는 문구의 화면에서는 "Close" 버튼으로 종료한다.

    만약 Python 사용에 문제가 있거나, third-party library를 설치하고 사용하는 데 문제가 있다면,
    현재 사용 중인 Python 소프트웨어를 삭제하고 여기 주어진 순서에 따라 Python 다시 설치.

  • Python 설치 확인
    • Python이 제대로 설치되었는 지 확인하는 것은 앞서 설명한 바와 같이 설치 마지막 화면을 통해서 얻을 수도 있으며,
    • 또는 새로 설치된 프로그램에서 Python을 실행시킴으로써도 얻을 수도 있다.
    • 하지만 무엇보다 Python 설치에 무엇보다 중요한 것은 Python의 외부 모듈을 설치하고 사용할 수 있는 지 여부이다.
      • 본 내용은 나중에 다시 살펴 보자.
    • 외부 모듈이란 PyPI (Python Package Index)를 통해 제3자가 만든 라이브러리 (또는 package, 몇 가지 예들은 numpy, scipy, pandas, matplotlib, 등등)를 사용하기 전에 반드시 설치해야 하는 것들이다.
    • 만약 앞서 언급한 Anaconda등과 같은 소프트웨어 등으로 Python을 설치한 경우는 해당 소프트웨어가 정해진 방식을 따라야 하겠지만, 앞서 언급한 9 단계를 통해 Python을 설치했다면 Python 설치할 때 함께 설치된 프로그램 중에 하나인 pip를 사용하는 것이다.
    • 앞서 언급한 9 단계 중 단계 8에서 선택한 내용이 해당 PC의 사용자들 모두에게 동일하도록 적용하도록 선택했기 때문에 외부 모듈 역시 설치할 때 유의사항이 있다.
    • 윈도우 시작 버튼을 누르고 Windows PowerShell(관리자) (A)를 선택한다.
      • 반드시 관리자 모드 이어야 한다.

    • 파란색 화면 시스템 prompt(>)에 pip list를 입력하고 <Enter>-키를 눌러 그동안 설치된 package들을 살펴본다.

      • 설치된 package들의 버전들을 볼 수 있다.
      • 만약 Python 설치가 제대로 되지 않았다면 오류가 나타날 것이다. 이 경우 그 동안 설치한 Python은 삭제하고 처음부터 다시 설치하라.
      • 표시된 내용 중 "WARNING: "으로 시작하는 문구가 보이면, 그 문구는 현재 설치된 pip 프로그램 (또는 앱)보다 최신 버전이 인터넷에 존재하니 update하라는 뜻이다.
        • 이 경우, 표시된 내용처럼 화면 시스템 prompt(>)에 표시된 내용 "python.exe -m pip install --upgrade pip" 그대로 적고 <Enter>-키를 누른다.
    • Windows PowerShell(관리자) (A) 화면은 나중에 외부 library (package)를 설치할 때도 필요하다. 기억해 두기를 권장한다.
      • 한 예로, 외부 모듈 numpy를 설치하려면, 시스템 prompt(>)에 "pip install numpy"라 입력하고 <Enter>-키를 눌러 실행한다.
      • 또 다른 예로, 외부 모듈 pandas를 설치하려면, 시스템 prompt(>)에 "pip install pandas"라 입력하고 <Enter>-키를 눌러 실행한다.
      • 외부 모듈 설치란 pip가 알아서 인터넷 상의 모듈을 찾아 내려 받은 후 설치과정을 거친다.
        • 대부분 성공적으로 설치되나, 그렇지 않은 경우도 있다. 이 경우는 경우마다 다르다 (죄송).

  • Python IDLE 시작하기
    • IDLE은 Integrated Development and Learning Environment의 약자
    • 셀(shell)창과 편집기(editor)창 두 가지로 구성
    • 셀 창
      • 작업 표시줄에 고정된 IDLE 아이콘을 클릭하면 셀 창이 실행된다.
      • 한 번에 한 줄의 소스 코드만을 실행하고 실행결과를 보여준다.
        • 2 줄 이상의 코드를 실행하면 오류 발생
      • 셀 창을 종료하면 그 동안 작업한 내용이 모두 사라진다.
      • 편집기 창에서 소스 코드들을 실행할 때 print()의 출력 결과를 보여주는 곳이기도 하다.
      • 편집기 창을 열고 싶으면 "메뉴/File/New"을 선택한다.
      • 예전에 편집하던 소스 파일을 열고 싶으면 "메뉴/File/Open"을 선택한다.
      • 최근 편집하던 소스 파일을 열고 싶으면 "메뉴/File/Recent"을 선택한다.
    • 편집기 창
      • 소스 코드들을 편집할 수 있다.
      • 소스 코드들을 실행하려면 반드시 메뉴판의 실행 버튼을 누르거나 함수키(F5)을 눌러야 한다.
      • 소스 코드를 파일로 저장하면 항상 원할 때 마다 편집할 수 있으며
      • 다른 Python 파일에서 불러서 사용할 수도 있다 (이 점이 바로 Python 언어의 가장 큰 특징)
      • 소스 코드를 실행하면 셀 창이 생성되며, 실행 결과는 셀 창에 표시된다.
      • 소스 코드 실행 없이 셀 창을 열고 싶으면 "메뉴/Run/Python Shell"을 선택한다.

  • Python 소스 파일 저장 폴더
    • 단순히 작업 표시줄에 저장된 IDLE icon을 클릭하고 새 파일을 만들어 저장하면, 기본 폴더는 Python 프로그램이 설치된 곳이다.
      • 이 곳 (Python 프로그램이 설치된 곳)에는 개인이 만든 Python source file 저장이 허락되지 않는다 (많은 입문자가 겪는 오류의 원인 중 한 가지).
    • 비록 PC를 본인만 사용하더라도 Python 코딩을 위해 폴더를 하나 정하고 그 폴더에 파일들을 저장하는 습관을 갖도록 노력하기를 권한다.
    • IDLE의 기본적으로 설정된 기능 중 한 가지는 편집된 파일을 실행할 때마다 파일을 저장할 것인지 묻는 창이 뜬다.
      • 편리한 기능이지만 여러 번 겪다보면 귀찮을 수 있으며,
      • 저장할 지 묻는 질문 대신 무조건 저장하도록 기능을 설정할 수 있다.
      • 아래 그림과 같이 "메뉴/Options/Configure IDLE/Settings/Shell/Ed"에서 현재 "Prompt to Save"에서 "No Prompt"로 바꿔 선택하고 저장한다.

    • Python 입문자에게 Python이 불편한 것 중 하나는 바로 폴더 변경에 관한 것이다.
    • 폴더 변경 방법은 여러 가지 가능하지만, 여기서 소개된 내용 모두가 서투른 상황 하에서 가장 쉬운 방법은 다음과 같다:
      • Shell 창에서 "메뉴/File/New"을 선택한 후 빈 Python 소스 파일을 저장한다.
        • 빈 소스 파일의 저장 위치 정할 때 폴더를 정한다.
    • 시간이 지나 어느 정도 Python의 다양한 기능이 익숙해지면 Python의 기본 모듈 os의 내장 함수 chdir()를 이용하여 폴더를 정할 수 있다.
